## Аннотация

Исследуется, в какой мере регуляризованная сумма по нетривиальным нулям ζ-функции способна предсказывать локальные колебания плотности простых чисел и может ли это давать практическое ускорение алгоритмов просеивания. Из явной формулы фон Мангольдта корректно выведен оператор плотности D(x,Λ) = −Σ_{γ≤Λ} ω(γ,Λ) cos(γ ln x); показано, что именно он (а не сумма +cos(γ ln x)/γ из прежней редакции) согласован по фазе и знаку с осцилляцией ψ′(x). На 2·10⁵ окнах в диапазоне x∈[2·10⁶, 9·10⁶] с настоящими простыми (сегментное решето) и настоящими нулями оператор D показывает статистически значимую, но микроскопическую положительную корреляцию с плотностью (r=+0.059, Z=+26 при Λ=200; R²≈0.0035). Оператор прежней редакции анти-коррелирует (r=−0.031, Z=−14): знак был перепутан. Оценка |S|=O(√N log N) воспроизводится, но оказывается тождеством счёта нулей Римана–фон Мангольдта при Λ=O(√N), а не свойством критерия отбора. Главный отрицательный результат: разметка «зон резонанса» НЕ даёт практического ускорения — доля захваченных простых при сканировании верхних k% окон совпадает с равномерной базой, а пропустить при потере <2% простых удаётся лишь ~2% окон (столько же, сколько даёт случайный порядок), а не заявленные 15–20%.

Ключевые слова: дзета-функция Римана, явная формула, плотность простых, спектральная эвристика, гауссова регуляризация, отрицательный результат.

## 1. Введение и постановка задачи

Нетривиальные нули ζ управляют распределением простых через явную формулу фон Мангольдта. Для функции Чебышёва ψ(x)=Σ_{pᵏ≤x} ln p:

```
ψ(x) = x − Σ_ρ x^ρ/ρ − ln(2π) − ½ ln(1 − x⁻²),   ρ = ½ + iγ.
```

Связь сумм по нулям с π(x) — прямое следствие этой формулы и не является самостоятельным открытием. Содержательный вопрос работы сугубо прикладной: можно ли по усечённой сумме нулей дёшево размечать интервалы повышенной плотности простых и получать за счёт этого выигрыш в просеивании. Ответ, как показано ниже, отрицательный, и важно понять — почему.

## 2. Спектральный оператор: корректное определение

### 2.1. Два разных объекта (исправление фазы и знака)

Прежняя редакция использовала U(x,Λ)=Σ ω(γ,Λ) cos(γ ln x)/γ с пояснением «φ=arg(ρ)=π/2, что соответствует φ=0». Это неверно. Разделим два объекта.

Счётная функция ψ. Вклад пары сопряжённых нулей в ψ(x)−x есть −2 x^{1/2} Re[e^{iγ ln x}/(½+iγ)] = −2 x^{1/2} [(½)cos(γ ln x)+γ sin(γ ln x)]/(¼+γ²), что при больших γ ведёт себя как −2 x^{1/2} sin(γ ln x)/γ. То есть для счётной функции правильная фаза — синус (а не косинус), с весом 1/γ.

Плотность простых. Дифференцируя явную формулу, получаем осцилляцию ПЛОТНОСТИ:

```
ψ′(x) = 1 − Σ_ρ x^{ρ−1} = 1 − 2 x^{−1/2} Σ_{γ>0} cos(γ ln x)  (при RH),
```

так что плотность простых вблизи x ≈ (1/ln x)·(1 − 2 x^{−1/2} Σ cos(γ ln x)). Здесь фаза — косинус, НО БЕЗ веса 1/γ, и простых локально БОЛЬШЕ там, где Σ cos(γ ln x) ОТРИЦАТЕЛЬНА. Определяем оператор плотности с гауссовым регулятором ω(γ,Λ)=exp(−γ²/2Λ²) и ведущим знаком минус, чтобы «больше D ⟺ больше простых»:

```
D(x,Λ) = − Σ_{γ_n ≤ Λ} ω(γ_n,Λ) · cos(γ_n ln x).
```

Оператор прежней редакции +cos/γ не совпадает ни со счётным (sin/γ), ни с плотностным (−cos без 1/γ) объектом; вес 1/γ делает его доминируемым несколькими низшими нулями, а знак для предсказания плотности оказывается перевёрнутым (см. 3.2).

### 2.2. Критерий отбора нулей по вкладу в дисперсию

Рабочее подмножество S(x,Λ)={ n : Varₙ(x) > 1/N }, где Varₙ — относительный вклад n-го нуля в локальную энергию на [x−δ, x+δ]: Varₙ = Eₙ/Σ_k E_k, Eₙ = mean_t [ω(γ_n) cos(γ_n ln t)]². Поскольку энергия монотонно убывает с γ (через ω и, в варианте с 1/γ, через 1/γ²), этот критерий по существу есть низкочастотный фильтр: он оставляет нули с наименьшими γ. Это полезно как удешевление, но не является x-адаптивным «резонансным» отбором.

### 2.3. Зоны конструктивного резонанса (исправление знака)

Зоной повышенной плотности следует называть интервал, где D(x,Λ) > μ_D + 1.5 σ_D (μ_D, σ_D — выборочные среднее и СКО на референсной сетке). Подчеркнём: в прежней редакции порог ставился на +U, тогда как из 2.1 следует, что высокая плотность отвечает отрицательной Σ cos, то есть высокому D=−Σ ω cos. Прежнее условие указывало в сторону МИНИМУМОВ плотности.

```
     N    Λ=c√N   #γ≤Λ    |S|    √N·lnN   N(Λ) РвМ
```

Колонка #γ≤Λ с точностью до единиц совпадает с N(Λ) Римана–фон Мангольдта (47 против 47.0). Значит «|S|=O(√N log N)» — это просто счёт нулей ниже отсечки Λ=O(√N): если Λ растёт как √N, то ниже неё лежит O(√N log N) нулей по теореме Римана–фон Мангольдта. Это тождество, а не свойство критерия Varₙ. При ФИКСИРОВАННОМ Λ=80 |S| насыщается на 6 (лишние нули выше отсечки, ω≈0) — никакого роста нет вовсе.

### 3.4. Перестановочный тест

Перемешивание мультимножества {γ} не меняет ни полную сумму (коммутативность), ни плотностный оператор (тот же набор частот). Поэтому перестановочный нуль на γ вырожден и не информативен сам по себе; в коде он допускается лишь как диагностика (флаг allow_degenerate_null / искусственный rank-фактор). Заявление прежней редакции «p<10⁻⁴ на 10⁴ перестановках» относилось к этому вырожденному построению и не несёт содержательного смысла.

## 4. Заявка на ускорение: проверка и опровержение

Главный практический вопрос: если размечать «зоны резонанса» по оператору и просеивать только их, какую долю простых мы захватим при какой доле просканированного континуума? Ранжируем 4·10⁴ окон по значению оператора и идём от высоких значений к низким.

Таблица 3. Доля захваченных простых при сканировании верхних k% окон.

```
  сканируем верхних:    D (−cos)    +cos/γ (стар.)   оракул(ист.плотн.)   случайно
        10%              9.98%        10.10%             11.11%            10.00%
        20%             20.01%        20.13%             21.77%            20.01%
        50%             50.10%        49.85%             52.50%            50.00%
        80%             80.03%        80.03%             81.72%            80.00%
```

Захват по любому оператору совпадает с равномерной базой (k% окон ⇒ ~k% простых). Даже ОРАКУЛ, ранжирующий по истинной плотности, при верхних 10% берёт лишь 11.1% простых — потолок концентрации на этом масштабе ничтожен. Контрольный вопрос: сколько окон можно пропустить, потеряв <2% простых?

```
    D (−cos):   можно пропустить  ≈2.0% окон
    +cos/γ:     можно пропустить  ≈2.0% окон
    случайно:   ≈2.0% окон  (то же самое)
```

Это ровно столько, сколько даёт случайный порядок, и несопоставимо с заявленными в прежней редакции 15–20%. Вывод: статистически эффект реален (Z=+26 для корректного D), но практически бесполезен — R²≈0.0035 не переводится в пригодную для просеивания концентрацию. Заявка на ускорение MPQS/GNFS на 15–20% не подтверждается и снимается.

## 5. Заключение

1) Корректно выведенный из явной формулы оператор плотности D=−Σ ω cos(γ ln x) действительно положительно и значимо коррелирует с локальной плотностью простых; оператор прежней редакции +cos/γ имел перепутанный знак и анти-коррелировал. 2) Эффект микроскопичен (R²≈0.0035) — это ожидаемое колебание амплитуды ~x^{−1/2}. 3) Оценка |S|=O(√N log N) — тождество счёта нулей при Λ=O(√N), а не результат отбора. 4) Практического ускорения просеивания нет: разметка зон эквивалентна случайной (пропуск ~2% при потере <2%). Ценность работы — методологическая и отрицательная: показано, как корректно строить спектральный предиктор плотности и почему он не даёт прикладного выигрыша.

Дальнейшее: проверка роста эффекта при x∼10¹⁴ и Λ∼10³ с настоящим сегментным решетом (эффект должен расти как функция числа нулей, оставаясь O(x^{−1/2}) по амплитуде); строгая оценка L²-ошибки усечения.
